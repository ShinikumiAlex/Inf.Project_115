Kotlin

Подробная информация о языке программирования Kotlin: история, особенности, преимущества, недостатки.

Примеры кода


fun main() {
    println("Hello, World!")
}

fun greet(name: String) {
    println("Привет, $name!")
}

fun init() {
    greet("Kotlin")
}
